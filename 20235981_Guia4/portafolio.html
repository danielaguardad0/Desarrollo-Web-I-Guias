Descripción del curso

Este curso es el primero en la secuencia de desarrollo web y cubre los fundamentos básicos de los lenguajes HTML, CSS y JavaScript, que son los elementos esenciales de cualquier aplicación web.

A través del dominio de los elementos del lenguaje HTML y el uso de los elementos de estilo en lenguaje CSS los estudiantes podrán diseñar sitios web atractivos y funcionales. Mediante el uso de los conceptos básicos y sintaxis de JavaScript, los estudiantes podrán crear aplicaciones interactivas de front-end e introducir contenido dinámico. Asimismo, serán capaces de comprender la arquitectura de las aplicaciones front-end, el rol de cada tecnología en la creación de aplicaciones modulares y conocerán las mejores prácticas en el diseño de estas. Finalmente, se discutirán algunas prácticas desventajosas en el desarrollo de software.

Retroalimentación sobre el curso

Guía 1

### Descripción de la Guía En esta guía, identificaras la estructura básica de un documento web, aplicará reglas de estilo CSS a elementos HTML mediante estilos embebidos y hojas externas, y utilizará selectores CSS básicos junto con propiedades para fuentes y textos.

Guía 2

En esta guía, aprenderas a utilizar y combinar selectores CSS avanzados, aprovechar pseudo-clases y pseudo-elementos, y crear páginas web aplicando el Modelo de Cajas (margin, border, padding, content).

Guía 3

En esta guía, aprenderás a utilizar y diferenciar los tipos de posicionamiento en CSS, aplicar propiedades de visualización de elementos, y maquetar sitios web con diseños fijos y fluidos.

Guía 4

Esta guía enseña a crear y estilizar formularios en HTML5, usar correctamente la meta-etiqueta `viewport` y aplicar media queries para adaptar el diseño según el ancho del navegador. Se pone en práctica el desarrollo de páginas web responsivas.

Retroalimentación sobre el curso

Lo que más me gustó del curso

Lo que más me gustó fue trabajar con los selectores CSS. Me dio el control necesario para estilizar diferentes elementos en función de sus clases, IDs o atributos específicos. El aprendizaje del modelo de caja fue esencial para entender cómo los márgenes, rellenos y bordes afectan el diseño y la disposición de los elementos.

Lo que menos me gustó del curso

Lo que encontré más desafiante fue cuando tratamos el diseño responsivo y las media queries. Aunque Flexbox facilitó el manejo de ciertos layouts, hacer que el diseño funcionara bien en todas las pantallas requería mucho tiempo y prueba.

Retos asumidos

El mayor reto fue el aprendizaje del posicionamiento y visualización en CSS, ya que hay muchas formas de hacerlo (absolute, relative, etc.) y cada una tiene sus particularidades. Resolver este reto fue un proceso de ensayo y error, además de buscar recursos externos como tutoriales para entender las mejores prácticas. También me costó dominar el modelo de caja, pero al practicar con varios ejemplos logré comprender cómo influye en la disposición de los elementos..

Sugerencias para mejorar

Sería útil incluir más ejercicios prácticos, para cubrir más a fondo ejercicios reales.
